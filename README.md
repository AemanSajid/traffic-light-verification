# Traffic Light Controller with Verification (Verilog FSM)
## Overview
This project implements a Traffic Light Controller using Verilog based on Finite State Machine (FSM) design and verifies it using a structured testbench.

## Features
- FSM-based traffic light (RED → GREEN → YELLOW)
- Timer-based transitions
- Pedestrian signal handling
- Emergency override

## Verification
- Reference (golden) model
- Scoreboard-based comparison
- Directed and random testing
